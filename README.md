# DOM Manipulation Menu Project


## Project Overview
This project demonstrates core JavaScript DOM manipulation concepts by dynamically building a webpage layout, navigation menu, and submenu entirely with JavaScript.

## Technologies Used
- HTML5
- CSS
- JavaScript (DOM Manipulation & Event Handling)

## Features
- Dynamically generated main content
- Top navigation menu created from a JavaScript data structure
- Submenu that appears based on selected menu item
- Event delegation for efficient click handling
- Active state toggling for menu items
- Data-driven submenu creation
- Clean separation of logic using helper functions
